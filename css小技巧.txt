/* add border */
.nav li {
  border-right: 1px solid #666;
}

//* remove border */
.nav li:last-child {
  border-right: none;
}

不过不要这么做，使用 :not() 伪类来达到同样的效果：

.nav li:not(:last-child) {
  border-right: 1px solid #666;
}
